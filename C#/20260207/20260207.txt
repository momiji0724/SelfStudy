testCode17:自動実装プロパティ
プロパティをすべて書くのが面倒な場合自動実装プロパティを使用する
読み書きを行うだけのプロパティは省略が可能
    public int MaxHp
    {
        set { _maxHp = value; }
        get { return _maxHp; }
    }
	↓
　　public int MaxHp { get; set; }
読み取り専用の場合はset;の前にprivateをつける

testCode18:参照と価
参照型はnewを作ったときだけ値が作られる
どこにあるのかという情報だけ取引される。
クラスは常に参照型になっている。
参照が存在しない場合nullとなり実行時にエラーが発生する

testCode19:構造体
構造体とはクラスとついになる値型である(structで書く)
参照型は複数の変数で同じものを共有することがある
値型はコピーされるので共有されることはない
クラスは参照型、構造体は常に値型であり、
構造体型のオブジェクトは代入でコピーされる
