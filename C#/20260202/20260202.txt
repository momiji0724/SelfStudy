testCode05:変数と型
変数には型があり、intは整数の型
変数の名前付けにはルールがあり、名前にマッチしている値を保存することができる
変数の名前被りはできないが、いつでも変数の値を設定し直すことはできる

testCode06:比較演算子
等値、比較演算で変数や値を調べることができる
== : 等しければ
!= : 等しくなければ
> : 大なり(右側の数字より大きい)
< : 小なり(右側の数字より小さい)
比較結果はtrueまたはfalseになる

testCode07:if文
if文によって処理を実行できるかどうかを選択できる
else節をもつif文は条件によって処理を分岐できる
if(条件[True or False判定])
{
	[条件がTrueだった場合表示したい文字]
}
else
{
	[条件がFalseだった場合表示したい文字]

}
あまりifを多用しすぎてコードを複雑にしすぎないのも大事
